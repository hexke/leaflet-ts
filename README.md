This project uses:
- pnpm [https://pnpm.io/],
- vite [https://vitejs.dev/],
- react [https://reactjs.org/],
-typescript [https://www.typescriptlang.org/]

installing project dependencies:
`pnpm install` 

starting dev server: 
`pnpm run dev`

server listens on:
`http://localhost:5173/`

generating build files:
`pnpm run build`
